# personal-finance-tracker 
A Python based application that allows users to track income and expenses.
The system stores data in an SQL database and generates basic summaries.

## Features
- Add and categorise income and expenses
- Store financial records using SQL
- Generate monthly summaries

## languages Used
- Python
- SQL

## Purpose
This project was developed to practise Python programming, database integration,
and problem-solving skills as part of my Software Engineering studies.
