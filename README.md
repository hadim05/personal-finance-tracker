# Personal Finance Tracker

A Python-based application that allows users to track income and expenses.
Data is stored using an SQL (SQLite) database.

## Features
- Add income and expenses
- Categorise transactions
- Store and retrieve data from a database
- View all financial records

## Technologies Used
- Python
- SQLite (SQL)

## How to Run
1. Download the files
2. Run `finance_tracker.py`
3. Follow the on-screen menu

## Purpose
This project was developed to practise Python programming, SQL database integration,
and problem-solving skills as part of my Software Engineering degree.
